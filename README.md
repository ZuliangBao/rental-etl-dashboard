# Rental Inquiry ETL Dashboard

## Project Overview

This project analyzes rental inquiry data from international students to understand housing demand, budget preferences, source channels, and conversion patterns.

The goal is to build a simple ETL pipeline, run SQL-based analysis, and design a dashboard that supports better rental recommendation and marketing decisions.

## Business Questions

1. Which neighborhoods receive the highest number of rental inquiries?
2. What budget ranges are most common among students?
3. Which source channels generate the most qualified leads?
4. What bedroom types are most requested?
5. How do rental inquiries change over time?
